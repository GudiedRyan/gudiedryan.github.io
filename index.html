About Me

I went to Allegheny College, where I majored in mathematics and minored in economics. During my last semester, I built a Python program for a professor that modeled the efficiency of a solar panel. As the pandemic began to close everything down, I found myself wondering what I was going to do with mathematics. When I spoke to a friend, he suggested I look into software engineering. Having done some coding in the past, I decided to give it a try and immediately fell in love with it. Ever since then I have been working on coding and seeking out new problems to solve. Aside from problem solving, I enjoy biking and Star Wars.

Guide and Teach is designed to help tutors organize their classes and students. I designed this program for tutors. This will enable teachers to track courses, students, and grades in an accessable manner. Click the link below to try it out!

Try it out!

Sudoku Solver

The sudoku solver is written in Python. Currently it can solve puzzles, provide hints, or tell you if the puzzle is solveable or not. The next step is to build a frontend to make it look presentable. This is a work in progress, so stay tuned for the final project!

Jeopardy Games

Ever wanted to play your own Jeopardy game without going on the show? Well this project will allow you to create your own Jeopardy games to play. No accounts required!

State Guessing Game

Using the turtles and pandas packages in Python, I built a state guessing game. Basically you are given a map of the United States but with none of the state names included. The states that the user guesses will appear, and the ones that the user does not guess will be collected and left in a file for the user to learn.

Matrix Inverter

The first React app I built and deployed was the Matrix Inverter. This allows you to put in any 2 x 2 or 3 x 3 matrix and find the inverse, given the inverse exists. It will also tell you if the inverse does not exist.

Try the Matrix Inverter

Lights Out

Another app build with React, Lights Out is a puzzle game where you try to turn off all the lights. There are two game boards available, 3x3 and 4x4.
